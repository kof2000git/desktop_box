# DesktopBox 桌面整理盒子 · 使用说明

> 一个常驻 Windows 桌面的「整理盒子」小工具。你可以把常用的程序、文件、文件夹、网址拖进一个个半透明的盒子里，让桌面变干净、找东西更快。类似 Fences / 360 桌面助手的桌面整理功能。

适用系统：**Windows 10 / Windows 11（64 位）**

---

## 一、两种安装方式（任选一种）

### 方式 A：安装包（推荐新手）
1. 双击运行 **`DesktopBoxSetup.exe`**。
2. 一路点 **Next → Next → Install → Finish**（安装向导是英文的，因为它用的是通用安装器，但软件本身是中文）。
3. 安装完会自动运行；之后也能在「开始菜单 → DesktopBox」里找到，或在安装目录里双击 `DesktopBox.exe`。
   - 默认安装到：`C:\Users\你的用户名\AppData\Local\Programs\DesktopBox`
   - **不需要管理员密码**（已设置成当前用户安装）。
   - 卸载：在「设置 → 应用」里找 DesktopBox 卸载，或用开始菜单里的「Uninstall DesktopBox」。

### 方式 B：绿色版（免安装）
1. 直接双击 **`DesktopBox.exe`** 即可运行，不写注册表、不装运行时（已经打包进去）。
2. 想放哪放哪，建议放在非系统盘的一个固定文件夹（例如 `D:\Software\DesktopBox\`），方便以后升级。
3. 想开机自启：运行后在「设置」里勾选「开机自启动」即可（会写入当前用户的注册表，无需管理员）。

## 二、第一次使用

程序启动后，你会看到桌面右下角任务栏多了一个**蓝色小方块图标**（托盘图标），桌面上可能还没有盒子。

### 1. 新建一个盒子
在**桌面空白处点鼠标右键** → 选 **「新建盒子」**。
（也可以在右下角托盘图标上点右键 →「新建盒子」。）

会出现一个半透明的深色卡片，名字默认是「盒子 1」。

### 2. 往盒子里放东西（拖放）
把**任意一个**下面的东西，用鼠标拖到盒子卡片上松手：
- **程序**（.exe，例如微信、浏览器）
- **文件**（文档、图片、压缩包……）
- **文件夹**
- **快捷方式**（.lnk，桌面/开始菜单里的那些）
- **网址**：从浏览器地址栏左边的小图标，或直接选中网址文字，拖到盒子上

放进去后会自动显示图标和名字。可以放很多个，盒子会自动换行排列。

### 3. 打开里面的东西
**双击**盒子里的图标，就会用系统默认方式打开它（程序会启动、文件会用对应软件打开、网址会用浏览器打开）。

## 三、盒子的常用操作

| 你想做…… | 怎么做 |
|---|---|
| 移动盒子位置 | **按住盒子上方标题栏**（写着名字的那一条）拖动 |
| 改变盒子大小 | 拖动盒子**右下角**的小方块 |
| 重命名盒子 | 在盒子上**点右键 →「重命名盒子」**，输入新名字 |
| 清空盒子 | 在盒子上**点右键 →「清空盒子」**（会先问你确认） |
| 删除整个盒子 | 在盒子上**点右键 →「删除盒子」**（会先问你确认） |
| 删除盒子里某个图标 | 在那个图标上**点右键 →「删除」** |
| 打开图标所在的文件夹 | 在图标上**点右键 →「打开所在位置」** |

> 所有改动都会**自动保存**，不用手动存。

## 四、托盘图标（右下角）

在任务栏右下角的蓝色小方块图标上：

- **双击**：显示 / 激活盒子
- **右键**弹出菜单：
  - 显示盒子
  - 新建盒子
  - 设置
  - 退出（**真正关闭程序**就靠这个）

> ⚠️ 直接点盒子窗口的关闭（×）**不会**退出程序，而是缩到托盘后台运行（这是为了盒子常驻桌面）。**要彻底退出，请在托盘图标上右键 →「退出」。**

---

## 五、设置

托盘右键 →「设置」，或在桌面空白处右键 →「设置」。

- **开机自启动**：勾上后，电脑开机会自动运行 DesktopBox。
- **跟随系统深浅色**：勾上后，盒子配色跟随 Windows 的深色/浅色模式。
- **主题 → 深色 / 浅色**：不跟随系统时，手动选一个。
  - 设置**即时生效并自动保存**，没有「确定」按钮。

## 六、我的数据存在哪？怎么备份？

所有盒子和条目保存在一个文件里：

```
C:\Users\你的用户名\AppData\Roaming\DesktopBox\boxes.json
```

图标缓存（自动生成）在：

```
C:\Users\你的用户名\AppData\Local\DesktopBox\icons\
```

**备份**：把 `boxes.json` 复制一份保存即可。换电脑/重装时，把这个文件放回同样位置就能恢复你的所有盒子。

> 注意：盒子保存的是「快捷方式的路径」，不是文件本身。如果你把原始文件删了或移动了，盒子里的图标点击时会打不开（这和普通快捷方式一样）。

## 七、常见问题（FAQ）

**Q：运行后桌面上什么都没有？**
A：盒子可能贴在桌面图标层。试试：① 在桌面空白处右键 →「新建盒子」；② 看右下角托盘有没有蓝色图标，双击它「显示盒子」。

**Q：按 Win+D（显示桌面）盒子不见了？**
A：盒子贴在桌面图标层，会和其它窗口一起被 Win+D 隐藏，再按一次 Win+D 或点托盘「显示盒子」即可。

**Q：盒子跑到别的窗口后面 / 看不到了？**
A：极少数情况下「贴桌面层」会失败，程序会自动降级为置顶窗口。点托盘「显示盒子」唤出。

**Q：拖进去的东西打不开？**
A：多半是原始文件被移动或删除了。在该图标上右键 →「打开所在位置」检查，或删掉重新拖一次。

**Q：有多块显示器，盒子只在主屏上？**
A：当前版本（MVP）只支持主显示器，后续版本会支持多屏。

**Q：杀毒软件拦截 / SmartScreen 提示？**
A：因为是自制的 exe 没有数字签名，Windows 可能弹 SmartScreen 警告。点「更多信息 → 仍要运行」即可。本程序是开源代码本地编译的，不联网、不上传任何数据。

**Q：怎么完全卸载？**
A：① 用安装包的话，在「设置 → 应用」里卸载 DesktopBox；② 然后手动删掉 `AppData\Roaming\DesktopBox` 和 `AppData\Local\DesktopBox` 两个文件夹以清理配置和图标缓存。

## 八、当前版本已知限制（MVP v1）

- 仅主显示器
- 网址条目使用默认图标（不抓网页图标）
- 盒子背景是半透明卡片模拟毛玻璃（系统级 Mica 与窗口透明无法同时启用）
- 安装向导界面为英文（软件本体全中文）
- 不操作真实桌面图标（只做独立整理盒，保证稳定不崩）

后续会加：多屏、网址图标、盒子折叠、搜索、自动归类导入等。

## 九、从源码自行编译（给进阶用户）

需要 **.NET 8 SDK** 和（可选）**Inno Setup 6**。

```bash
# 还原 + 构建 + 测试
dotnet build DesktopBox.sln
dotnet test DesktopBox.sln

# 发布绿色单文件 exe
dotnet publish src/DesktopBox -c Release -r win-x64 \
  --self-contained true \
  -p:PublishSingleFile=true \
  -p:IncludeNativeLibrariesForSelfExtract=true \
  -p:EnableCompressionInSingleFile=true \
  -o publish

# （可选）编译安装包
"C:\Users\<你>\AppData\Local\Programs\Inno Setup 6\ISCC.exe" DesktopBox.iss
```
