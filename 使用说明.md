# DesktopBox 桌面整理盒子 · 使用说明

> 常驻 Windows 桌面的「整理盒子」工具。可把程序/文件/文件夹/网址拖进盒子分类；也能**一键把桌面散乱文件自动分类，合并到一个带标签页的盒子**（**只引用、不移动**，其它程序照常工作）；还能隐藏桌面图标、用盒子存放回收站等系统图标。类似 Fences / 360 桌面助手。

适用系统：**Windows 10 / Windows 11（64 位）**

---

## 一、安装（任选一种）

### 方式 A：安装包
双击 **`DesktopBoxSetup.exe`** → 一路 Next → Install → Finish（向导英文，软件本体中文）。装到当前用户目录，**不需管理员密码**。卸载：「设置 → 应用」里卸载 DesktopBox。

### 方式 B：绿色版
双击 **`DesktopBox.exe`** 即可运行，免安装、不写注册表。开机自启在「设置」里勾选。

---

## 二、手动使用

启动后右下角有**蓝色小方块**托盘图标。

- **新建盒子**：桌面空白处右键 / 托盘右键 →「新建盒子」。
- **放东西**：把程序/文件/文件夹/快捷方式/网址**拖**到盒子上，自动显示图标和名字。
- **打开**：**双击**图标。
- **盒子操作**：按住标题栏拖动移动；**拖盒子四边或四角缩放**；内容多时**自动滚动条**；右键有 重命名/清空/删除盒子。
- **图标操作**：图标右键弹出**原生 Windows 右键菜单**（打开 / 属性 / 发送到 / …，与资源管理器一致），末尾还有「从盒子移除」。

> 所有改动自动保存。点窗口 × 是缩到托盘；**彻底退出**在托盘右键 →「退出」。

---

## 三、⭐ 一键整理桌面（增量、只引用、不移动）

**入口**：桌面空白处右键 / 托盘右键 → **「一键整理桌面」**。

**它做什么**：扫描桌面 → 按类型自动归类，**合并到一个带标签页的盒子「桌面整理」**（每个分类是一个标签）：**应用程序 / 文档 / 图片 / 视频 / 音频 / 快捷方式 / 压缩包 / 文件夹 / 其他**。盒子**引用**桌面文件的路径。

### 🔄 增量整理
- **第一次**整理会创建「桌面整理」标签盒子，把所有桌面文件分类进去，**并自动添加一个「系统图标」标签**（此电脑 / 回收站 / 控制面板 / 网络）。
- **以后每次**整理只处理"还没进盒子"的**新文件**，按类型自动归到对应标签；没有新文件时提示"已全部整理"。
- 没有"还原"概念——整理是持续维护，文件从不移动。

> 💡 想清空重来：右键整理盒子 →「清空盒子」→ 再「删除盒子」；下次整理会重新建。

### ✅ 关键：文件不会被移动
- 文件**原地不动**，写死桌面路径的其它程序/脚本照常工作（这是相对"移动式整理"的最大优点）。
- 代价：桌面上**真实图标还在**。想让桌面干净，用下面的「隐藏桌面图标」。

---

## 三·补、🗂️ 标签盒子（把多个盒子合并到一处）

除了「一键整理」会自动生成标签盒子，你也可以手动把任意盒子变成标签盒子、或合并/拆分（**盒子右键**）：

- **转为标签盒子**：把普通盒子的内容收进首个标签，变成标签盒子。
- **合并其它盒子进来…**：选另一个盒子，它作为**新标签**并入当前盒子（原盒子消失，内容不丢）。
- **拆分为独立盒子**：标签盒子的每个标签还原成独立盒子（位置略有错开，可拖开）。

**标签操作**：
- 点顶部标签切换显示；**按住标签拖动**可调整顺序；标签行末「**+**」新建空标签。
- **盒子右键** →「新建标签 / 重命名当前标签 / 删除当前标签」（空标签才能删，最后一个标签不能删）。
- 把文件/网址**拖**进标签盒子 → 加到**当前选中**的标签。

> 适合桌面盒子太多、想合并省空间时用。文件始终不被移动。

---

## 四、🖼️ 隐藏 / 显示桌面图标

**入口**：桌面空白处右键 / 托盘右键 → **「隐藏/显示桌面图标」**。

- 点一下**隐藏**所有桌面真实图标（只剩我们的盒子），桌面瞬间干净。
- 再点一下**显示**，恢复。
- **纯视觉开关，不移动任何文件**；其它程序的路径引用完全不受影响。

> ⚠️ 这是 Windows 的全局开关，隐藏时**回收站、此电脑等系统图标也会一起隐藏**。它们仍可用——见下面的「系统图标盒子」。再次点击即可全部恢复。

---

## 五、🖥️ 系统图标盒子（回收站/此电脑/控制面板/网络）

**入口**：桌面空白处右键 / 托盘右键 → **「添加系统图标盒子」**。

- 自动建一个「系统图标」盒子，里面放：**此电脑、回收站、控制面板、网络**（用真实系统图标）。
- **双击**即可打开对应的系统项。
- 适合配合「隐藏桌面图标」用：桌面图标隐藏后，回收站等仍能从盒子里点开。
- 已存在系统图标盒子时再点会提示，不重复创建。

> 这些是 Windows 系统图标（Shell 虚拟项），不是真实文件；从盒子里删除它们只是移除显示，不影响系统本身。

---

## 六、托盘菜单

右键托盘蓝色图标：显示盒子 / 新建盒子 / **添加系统图标盒子** / **一键整理桌面** / **隐藏/显示桌面图标** / 设置 / 退出。双击 = 显示盒子。

## 七、设置

托盘或桌面右键 →「设置」：**开机自启动** / **跟随系统深浅色** / **主题 深色·浅色**。即时生效并自动保存。

## 八、数据在哪？怎么备份？

**绿色便携版**：所有数据都放在 **可执行文件同目录**，删除 exe 即彻底清除，搬走 exe 即备份。

| 内容 | 位置 |
|---|---|
| 盒子配置（几 KB） | `DesktopBox.exe 同目录\boxes.json` |
| 整理记录（几 KB） | `DesktopBox.exe 同目录\organize.json` |
| 图标缓存 | `DesktopBox.exe 同目录\icons\` |
| 错误日志 | `DesktopBox.exe 同目录\logs\error.log` |

**备份盒子**：复制 `boxes.json` 即可。

> 本版本**不移动任何文件**，桌面文件始终在原处。盒子里存的是文件「路径」；你手动删/移走原始文件后，图标会打不开（和普通快捷方式一样）。

---

## 九、安全说明

- 程序**永远不会删除或复制你的文件**（整理是"只引用"，不移动）。
- 唯一的 `File.Delete` 只删程序自己的 `organize.json` 记录文件。
- 界面删除盒子/图标只删配置引用，文件本体不动。
- 非空盒子**不允许直接删除**（防止误以为文件丢失）。
- "隐藏桌面图标"只是视觉开关，不动文件。

## 十、常见问题

**Q：运行后桌面上什么都没有？**
桌面空白处右键 →「新建盒子」；或双击托盘蓝色图标「显示盒子」。

**Q：一键整理后文件去哪了？**
**哪也没去，文件还在桌面原处。**整理只是按分类把它们"列进盒子"。想干净用「隐藏桌面图标」。

**Q：整理会不会弄丢文件/影响别的程序？**
不会。只引用路径，文件不动，写死桌面路径的程序照常工作。

**Q：隐藏桌面图标后回收站没了？**
用「添加系统图标盒子」，回收站/此电脑/控制面板都在盒子里能点开。或再点「隐藏/显示桌面图标」恢复。

**Q：为什么不能删除非空盒子？**
为防止误以为文件丢失。先「清空盒子」或逐个删图标，再删；整理生成的盒子用「还原整理」。

**Q：Win+D 盒子不见了？**
盒子贴桌面图标层，随 Win+D 隐藏，再按或点托盘「显示盒子」。

**Q：多显示器？**
当前仅主显示器，后续支持多屏。

**Q：SmartScreen 拦截？**
点「更多信息 → 仍要运行」。本程序不联网、不上传任何数据。

**Q：完全卸载？**
**绿色版**：直接删除 exe 及同目录的 `boxes.json` / `organize.json` / `icons` / `logs` 即彻底清除。**安装版**：「设置 → 应用」卸载 DesktopBox。（文件都在桌面原处，无需处理。）

## 十二、从源码编译（进阶）

需 .NET 8 SDK（C# 主程序）、MSVC / Visual Studio Build Tools（编译原生右键菜单 DLL），可选 Inno Setup 6（打安装包）。详见 [README.md](README.md#-build-from-source--从源码编译)。

```bash
dotnet build DesktopBox.sln
dotnet test DesktopBox.sln     # 21 项单测
dotnet publish src/DesktopBox -c Release -r win-x64 --self-contained true \
  -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true \
  -p:EnableCompressionInSingleFile=true -o publish
# 原生右键菜单 DLL（在「x64 本机工具命令提示符」中运行）：
src\DesktopBox.ShellMenu\build_dll.bat
# 安装包（可选）：
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" DesktopBox.iss
```
