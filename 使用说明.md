# DesktopBox 桌面整理盒子 · 使用说明

> 常驻 Windows 桌面的「整理盒子」工具。可把程序/文件/文件夹/网址拖进盒子分类；也能**一键把桌面散乱的文件自动分类整理进盒子**，让桌面瞬间干净。类似 Fences / 360 桌面助手。

适用系统：**Windows 10 / Windows 11（64 位）**

---

## 一、安装（任选一种）

### 方式 A：安装包
双击 **`DesktopBoxSetup.exe`** → 一路 Next → Install → Finish（向导英文，软件本体中文）。安装到当前用户目录，**不需管理员密码**。卸载：「设置 → 应用」里卸载 DesktopBox。

### 方式 B：绿色版
双击 **`DesktopBox.exe`** 即可运行，免安装、不写注册表。建议放固定文件夹方便升级。开机自启在「设置」里勾选。

---

## 二、手动使用

程序启动后，右下角任务栏有**蓝色小方块**托盘图标。

- **新建盒子**：桌面空白处右键 / 托盘右键 →「新建盒子」。
- **放东西**：把程序/文件/文件夹/快捷方式/网址**拖**到盒子卡片上，自动显示图标和名字。
- **打开**：**双击**图标。
- **盒子操作**：按住标题栏拖动移动；拖右下角小方块缩放；内容多时**自动出现滚动条**；右键有重命名/清空/删除盒子。
- **图标操作**：图标右键有 打开 / 打开所在位置 / 删除。

> 所有改动自动保存。点窗口 × 是缩到托盘（盒子要常驻）；**彻底退出**在托盘右键 →「退出」。

---

## 三、⭐ 一键整理桌面

**入口**：桌面空白处右键 / 托盘右键 → **「一键整理桌面」**。

**它会做什么：**
1. 扫描桌面所有文件/文件夹/快捷方式。
2. 按类型自动归类并**自动建盒**，盒子名 = 分类名：
   **程序 / 文档 / 图片 / 压缩包 / 视频 / 音乐 / 文件夹 / 其他**
3. 把桌面文件**移动**到一个专门的整理文件夹，桌面就干净了。
4. 只有有内容的分类才建盒。

> 🚀 **很快**：整理文件夹**与桌面在同一个盘上**，移动只是改目录项（瞬间完成，文件再大也一样），**不占用 C 盘**。整理后弹「完成」提示；图标随后在后台加载（图标提取是最慢的环节，放后台不影响整理速度）。

### 文件被移到哪了？
整理文件夹的位置 = **桌面所在盘、与桌面同级的 `DesktopBox整理` 文件夹**。
你的桌面在 `K:\DESKTOP\Desktop`，所以整理文件夹是：

```
K:\DESKTOP\DesktopBox整理\
    ├── 程序\   ├── 文档\   ├── 图片\   ├── 压缩包\
    ├── 视频\   ├── 音乐\   ├── 文件夹\ └── 其他\
```

**不在 C 盘，不在桌面上**（放桌面就白整理了）。

> ⚠️ 整理会**移动**（不是删除、不是复制）你的文件。移动前会弹确认框显示数量和目标路径。**任何时刻都能用「还原整理」撤销**。

### 还原整理（一键撤销）
桌面右键 / 托盘右键 → **「还原整理」**：把所有文件移回桌面原位、删除自动生成的盒子（手动建的盒子不受影响）。

> 连续整理前必须先「还原」上一次。

## 四、托盘菜单

右键托盘图标：显示盒子 / 新建盒子 / 一键整理桌面 / 还原整理 / 设置 / 退出。双击 = 显示盒子。

## 五、设置

- **开机自启动**
- **跟随系统深浅色**（盒子配色跟系统）
- **主题 → 深色 / 浅色**

设置即时生效并自动保存。

---

## 六、数据存在哪？

| 内容 | 位置 |
|---|---|
| 盒子配置（很小） | `C:\Users\你\AppData\Roaming\DesktopBox\boxes.json` |
| 整理记录（很小） | `C:\Users\你\AppData\Roaming\DesktopBox\organize.json` |
| **整理移过去的文件** | **`K:\DESKTOP\DesktopBox整理\`**（与桌面同盘，不占 C 盘） |
| 图标缓存 | `C:\Users\你\AppData\Local\DesktopBox\icons\` |

> 配置文件(boxes.json/organize.json)只有几 KB，放 C 盘没问题；**真正占空间的是整理后的文件，它们在 K 盘**。

**备份盒子**：复制 `boxes.json` 即可。注意盒子里存的是文件「路径」；原始文件被你手动删/移走后，图标会打不开（和普通快捷方式一样）。

---

## 七、安全说明（重要）

- 程序**只会移动文件，永远不会删除或复制你的文件**。
- 唯一的 `File.Delete` 只作用于程序自己的 `organize.json` 记录文件，不碰你的数据。
- 在界面上「删除盒子/删除图标」**只删除配置里的引用，文件本体原样留在磁盘**。
- 整理前必须确认；被占用/无权限的文件会被跳过（不影响其它文件）。
- 万一程序异常，文件也在 `K:\DESKTOP\DesktopBox整理\` 里，可手动取回。

## 八、常见问题

**Q：运行后桌面上什么都没有？**
桌面空白处右键 →「新建盒子」；或双击托盘蓝色图标「显示盒子」。

**Q：一键整理后文件去哪了？**
`K:\DESKTOP\DesktopBox整理\` 下按分类存放。在盒子图标右键 →「打开所在位置」直达；或「还原整理」移回桌面。

**Q：整理会不会弄丢文件？**
不会，只是移动，原始数据完整，「还原整理」可移回。

**Q：整理很慢吗？**
不慢。文件在同盘(K:)内移动是瞬间的；图标在后台加载，不卡整理。

**Q：Win+D 盒子不见了？**
盒子贴桌面图标层，随 Win+D 隐藏，再按一次或点托盘「显示盒子」。

**Q：多显示器？**
当前版本仅主显示器，后续支持多屏。

**Q：SmartScreen 拦截？**
点「更多信息 → 仍要运行」。本程序不联网、不上传任何数据。

**Q：完全卸载？**
「设置 → 应用」卸载 DesktopBox；再手动删 `AppData\Roaming\DesktopBox`、`AppData\Local\DesktopBox`，以及 `K:\DESKTOP\DesktopBox整理`（如果还要保留里面的文件请先取出）。

## 九、当前版本已知限制（MVP）

- 仅主显示器
- 网址条目用默认图标
- 盒子背景是半透明卡片模拟毛玻璃（系统级 Mica 与透明窗口无法同时启用）
- 安装向导英文（软件本体中文）
- 不直接操作 explorer 桌面图标列表，只移动文件 + 显示独立盒子，保证稳定

后续：多屏、网址图标、盒子折叠、搜索等。

## 十、从源码编译（进阶）

需 .NET 8 SDK（可选 Inno Setup 6）。

```bash
dotnet build DesktopBox.sln
dotnet test DesktopBox.sln     # 30 项单测
dotnet publish src/DesktopBox -c Release -r win-x64 --self-contained true \
  -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true \
  -p:EnableCompressionInSingleFile=true -o publish
"C:\Users\<你>\AppData\Local\Programs\Inno Setup 6\ISCC.exe" DesktopBox.iss
```

## 📌 更新说明

### v1.2 删除盒子保护
- **非空盒子不允许直接删除**：右键「删除盒子」时,若盒子里还有内容,会弹提示拒绝删除——防止你误以为文件丢失。
- 要删除非空盒子,请先「清空盒子」(或逐个删图标),再删;一键整理生成的盒子想恢复桌面,请用「还原整理」。
- 删除盒子**永远不会删文件**:只是移除显示。整理来的文件仍留在 `K:\DESKTOP\DesktopBox整理\`。

### v1.1 稳定性
- 修复「删除盒子导致程序退出」的 bug(改用显式退出模式,只有托盘「退出」才会结束程序)。
