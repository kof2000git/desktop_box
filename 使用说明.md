# DesktopBox 桌面整理盒子 · 使用说明

> 一个常驻 Windows 桌面的「整理盒子」小工具。可以把常用的程序、文件、文件夹、网址拖进一个个半透明的盒子里；也能**一键把桌面上散乱的文件自动分类整理进盒子**，让桌面瞬间干净。类似 Fences / 360 桌面助手的桌面整理功能。

适用系统：**Windows 10 / Windows 11（64 位）**

---

## 一、两种安装方式（任选一种）

### 方式 A：安装包（推荐新手）
1. 双击运行 **`DesktopBoxSetup.exe`**。
2. 一路点 **Next → Install → Finish**（安装向导是英文的，因为用的是通用安装器，但软件本身是中文）。
3. 安装完会自动运行；之后在「开始菜单 → DesktopBox」或安装目录里双击 `DesktopBox.exe`。
   - 默认安装到：`C:\Users\你的用户名\AppData\Local\Programs\DesktopBox`
   - **不需要管理员密码**。
   - 卸载：「设置 → 应用」里卸载，或开始菜单「Uninstall DesktopBox」。

### 方式 B：绿色版（免安装）
1. 双击 **`DesktopBox.exe`** 即可运行，不写注册表、不装运行时（已打包进去）。
2. 建议放在固定文件夹（如 `D:\Software\DesktopBox\`），方便升级。
3. 开机自启：运行后在「设置」里勾选「开机自启动」。

---

## 二、第一次使用 / 手动整理

程序启动后，桌面右下角任务栏多了一个**蓝色小方块图标**（托盘）。

### 1. 新建盒子
在**桌面空白处点右键** →「新建盒子」（或托盘图标右键 →「新建盒子」）。会出现一个半透明深色卡片。

### 2. 往盒子里放东西（拖放）
把下面的东西拖到盒子卡片上松手：程序(.exe)、文件、文件夹、快捷方式(.lnk)、网址。
放进去后自动显示图标和名字，可放很多个，自动换行。

### 3. 打开里面的东西
**双击**图标即用系统默认方式打开。

## 三、⭐ 一键整理桌面（自动分类）

嫌手动拖太麻烦？桌面上堆了一堆文件？用这个：

**入口**（三处任选）：
- 桌面空白处**右键 →「一键整理桌面」**
- 托盘图标**右键 →「一键整理桌面」**

**它会做什么：**
1. 扫描桌面上的所有文件、文件夹、快捷方式。
2. 按类型**自动归类**，并**自动生成盒子**，盒子名字就是分类名：
   - **程序**（.exe / 快捷方式指向程序）
   - **文档**（.pdf .doc .docx .xls .xlsx .ppt .txt …）
   - **图片**（.jpg .png .gif .bmp .webp …）
   - **压缩包**（.zip .rar .7z …）
   - **视频**（.mp4 .avi .mkv …）
   - **音乐**（.mp3 .wav .flac …）
   - **文件夹**
   - **其他**（不属于以上）
3. 只有「确实有内容」的分类才会建盒子。
4. 整理时，会把桌面上的原始文件**移动**到：
   `C:\Users\你的用户名\AppData\Roaming\DesktopBox\Organized\分类名\`
   —— 这样桌面才会真正变干净。
5. 整理完会提示创建了几个盒子、整理了几个项目。

> ⚠️ **重要：整理会移动你的文件**。移动前会先弹确认框，告诉你数量和目标路径。**任何时刻都可以用「还原整理」撤销**（见下）。

### 还原整理（一键撤销）
- 桌面右键 / 托盘右键 →「**还原整理**」
- 会把所有已整理的文件**移回桌面原位**，并**删除自动生成的盒子**。
- 手动新建的盒子不受影响。

> 小贴士：连续整理前必须先「还原整理」上一次的操作。整理记录保存在 `AppData\Roaming\DesktopBox\organize.json`。

## 四、盒子的常用操作

| 你想做…… | 怎么做 |
|---|---|
| 移动盒子 | **按住标题栏**（盒子顶部名字那一条）拖动 |
| 改变大小 | 拖动盒子**右下角**小方块 |
| 盒子内容太多看不全 | **自动出现滚动条**，可上下滑动（把盒子拖大也能看更多） |
| 重命名盒子 | 盒子上**右键 →「重命名盒子」** |
| 清空盒子 | 盒子上**右键 →「清空盒子」** |
| 删除盒子 | 盒子上**右键 →「删除盒子」** |
| 删除某个图标 | 图标上**右键 →「删除」** |
| 打开图标所在文件夹 | 图标上**右键 →「打开所在位置」** |

> 所有改动**自动保存**，无需手动存。

## 五、托盘图标（右下角）

任务栏右下角蓝色小方块：

- **双击**：显示 / 激活盒子
- **右键**菜单：显示盒子 / 新建盒子 / 一键整理桌面 / 还原整理 / 设置 / 退出

> ⚠️ 点盒子窗口的关闭（×）**不会**退出程序，而是缩到托盘后台（盒子要常驻桌面）。**彻底退出请在托盘图标右键 →「退出」。**

---

## 六、设置

托盘或桌面右键 →「设置」：

- **开机自启动**：勾上后开机自动运行。
- **跟随系统深浅色**：盒子配色跟随 Windows 深色/浅色。
- **主题 → 深色 / 浅色**：不跟随系统时手动选。
  - 设置**即时生效并自动保存**。

## 七、数据存在哪？怎么备份？

| 内容 | 位置 |
|---|---|
| 盒子与条目配置 | `C:\Users\你的用户名\AppData\Roaming\DesktopBox\boxes.json` |
| 整理记录(还原用) | `C:\Users\你的用户名\AppData\Roaming\DesktopBox\organize.json` |
| 一键整理移过去的文件 | `C:\Users\你的用户名\AppData\Roaming\DesktopBox\Organized\` |
| 图标缓存 | `C:\Users\你的用户名\AppData\Local\DesktopBox\icons\` |

**备份**：复制 `boxes.json` 即可。换电脑时放回同样位置即可恢复你的盒子。

> 注意：盒子里存的是文件「路径」，不是文件本身。如果你手动删了或移走了原始文件，那个图标会打不开（和普通快捷方式一样）。

## 八、常见问题（FAQ）

**Q：运行后桌面上什么都没有？**
A：盒子贴在桌面图标层。① 桌面空白处右键 →「新建盒子」；② 看右下角托盘有没有蓝色图标，双击它「显示盒子」。

**Q：一键整理后，文件去哪了？**
A：被移到了 `AppData\Roaming\DesktopBox\Organized\` 下按分类存放。在盒子图标上右键 →「打开所在位置」就能找到。或用「还原整理」移回桌面。

**Q：一键整理会不会弄丢文件？**
A：不会。只是「移动」到整理文件夹，原始数据完整保留。「还原整理」可一键移回。万一程序异常，文件也在那个文件夹里，可手动取回。

**Q：为什么有些文件没被整理进去？**
A：被其它程序占用、或权限不足的文件会被跳过（不会影响其它文件）。

**Q：按 Win+D 盒子不见了？**
A：盒子贴桌面图标层，会随 Win+D 一起隐藏，再按一次或点托盘「显示盒子」即可。

**Q：拖进去的东西打不开？**
A：多半原始文件被移动/删除了。在该图标右键 →「打开所在位置」检查。

**Q：多显示器？**
A：当前版本仅主显示器，后续支持多屏。

**Q：SmartScreen 拦截？**
A：自制 exe 无数字签名，Windows 可能提示。点「更多信息 → 仍要运行」。本程序不联网、不上传任何数据。

**Q：完全卸载？**
A：① 「设置 → 应用」卸载 DesktopBox；② 手动删 `AppData\Roaming\DesktopBox` 和 `AppData\Local\DesktopBox` 两个文件夹。

## 九、当前版本已知限制（MVP）

- 仅主显示器
- 网址条目使用默认图标
- 盒子背景是半透明卡片模拟毛玻璃（系统级 Mica 与透明窗口无法同时启用）
- 安装向导界面英文（软件本体全中文）
- 不直接操作 explorer 的桌面图标列表（只移动文件、显示独立盒子），保证稳定不崩

后续会加：多屏、网址图标、盒子折叠、搜索等。

## 十、从源码自行编译（进阶）

需要 **.NET 8 SDK** 和（可选）**Inno Setup 6**。

```bash
dotnet build DesktopBox.sln
dotnet test DesktopBox.sln     # 30 项单测

dotnet publish src/DesktopBox -c Release -r win-x64 \
  --self-contained true \
  -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true \
  -p:EnableCompressionInSingleFile=true -o publish

"C:\Users\<你>\AppData\Local\Programs\Inno Setup 6\ISCC.exe" DesktopBox.iss
```
